# CostPilot

AI SaaS spend audit platform for startups.

## Problem

Startups overspend on AI tools because:
- duplicate subscriptions
- unused seats
- wrong plans
- overlapping products
- expensive models used unnecessarily

## Solution

CostPilot analyzes:
- AI tools
- pricing plans
- seat counts
- monthly spend
- usage patterns

Then recommends:
- cheaper alternatives
- seat reductions
- plan optimizations
- tool consolidation
- potential savings

---

## MVP Features

### Input
- Tool name
- Monthly spend
- Seats
- Usage type
- Team size

### Analysis
- Detect overspending
- Detect overlapping tools
- Suggest cheaper plans
- Suggest alternative tools
- Estimate savings

### Output
- Total monthly AI spend
- Waste estimate
- Optimization suggestions
- Savings potential
- AI spend score
